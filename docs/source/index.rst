Welcome to flask-socketio-lit-html's documentation - Flask Web Element Project
===================================================
**F**\lask-**W**\eb-**el**\ement - abbreviated Flask-Wel - *Webcomponents with Flask in Python !*

This project was first design to quickly get a web app with one unique stack. No need to set-up a nodejs, PHP frameworks, with Angular, Vue or other frontends. This alpha version 0.1.x is a first draft of an opinionated flask library. Reuse still in mind, Flask-Wel components should be easily integrated or rewrited in any other stack.
